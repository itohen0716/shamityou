# 絲庵 ─ Shamisen ─ Ver.2.0

GitHub Pagesへ、そのままアップロードできるフォルダです。

## アップロード方法

1. このZIPを展開します。
2. 中にあるファイルとフォルダを、GitHubリポジトリの最上位へアップロードします。
3. GitHub Pagesを `main` / `(root)` に設定します。
4. 公開URLの `tuning.html` を開き、マイク使用を許可します。

## 搭載内容

- 本調子・二上り・三下り
- 一の糸 1本～12本
- ① 三弦を順番に合わせる
- ② 一音、一音合わせる
- お師匠さんのWAV音源を基準音に使用
- 日本語による高低判定
- 三弦連続モードの自動進行
- シャミの場面画像・表情画像
- PWA、オフラインキャッシュ

## 重要

スマートフォン自身の基準音をマイクが拾うため、基準音再生中と直後は判定を短時間休止します。
実機や録音環境により判定幅の微調整が必要になる場合があります。

元音源:
`sounds/teacher-1to12-octave.wav`
