# erika

GitHub Pages用の「協力・監修｜染行エリカ」ページです。

## ファイル構成

- `index.html`
- `style.css`
- `script.js`
- `Image/erika-placeholder.svg`

## 設置方法

1. ZIPを解凍します。
2. `erika`フォルダを、調弦アプリのルートフォルダへアップロードします。
3. 調弦アプリ側のボタンやリンクから、`erika/index.html`へ移動するよう設定します。

## プロフィール画像の差し替え

実際の写真を使用する場合は、画像を `Image/erika.jpg` として保存し、
`index.html` 内の次の部分を変更してください。

`src="Image/erika-placeholder.svg"`

↓

`src="Image/erika.jpg"`

## ホームへ戻るボタン

`../index.html` に戻る設定です。
調弦アプリ本体と `erika` フォルダが同じ階層にある場合、そのまま使用できます。
