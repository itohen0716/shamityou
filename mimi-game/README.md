# 耳を育てるゲーム

## 設置方法

既存の調弦アプリ直下に、この `game` フォルダをそのまま配置してください。

```text
調弦アプリ/
├── index.html
├── ...
└── game/
    ├── index.html
    ├── game.css
    ├── game.js
    └── images/
```

ホーム画面の「耳を育てる」ボタンのリンク先を次にします。

```html
<a href="./game/index.html">耳を育てる</a>
```

## 現在の仕様

- 調子を当てる
  - 基本：本数を表示
  - 応用：本数も回答
- 音を合わせる
  - 1問20秒
  - 10問
  - ±5以内を正解
- 正解数、連続正解数、最高連続正解数
- シャミ画像12種類
- マイク不使用
- 音はWeb Audio APIで仮生成

## 重要

現在の本数と音高の対応は仮設定です。

`game.js` の次の値を、既存調弦アプリの設定に合わせて変更できます。

```javascript
const HON_BASE_MIDI = 48;
```

調子の音程関係は次の設定です。

```javascript
本調子:  [0, 5, 12]
二上り:  [0, 7, 12]
三下り:  [0, 5, 10]
```
